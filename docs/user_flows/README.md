# User Flows

Esta carpeta contiene diagramas de flujo que describen la interacción del usuario paso a paso para completar las funcionalidades principales (Happy Paths y flujos alternativos).

## Flujos Documentados

### 1. Onboarding y Creación de Horario
- **Archivo**: [`01_onboarding_flow.puml`](01_onboarding_flow.puml)
- **Descripción**: Proceso desde que el usuario define su semestre hasta que carga sus materias, manejando validaciones y conflictos.

### 2. Gestión de Tareas (Próximamente)
- Creación de tarea -> Asignación de materia -> Sincronización GCal.

## Cómo agregar nuevos flujos
1. Crear un archivo `.puml` en este directorio.
2. Usar sintaxis de diagrama de actividad de PlantUML.
3. Registrar el flujo en este README.
